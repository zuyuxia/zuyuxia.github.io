map与unordered_map

发表于 2018-03-30 | 分类于 STL

map

map是一种关联容器，里面的元素是key-value键值对。关键字key起到索引的作用，值value表示与索引key相关联的数据。map底层的实现是一棵红黑树，对数据具有自动排序的功能，因此在map内的数据都是有序的

map的操作（增删查改）

  • map[key] = value; map.insert(pair<key,value>(key,value));
  • map.erase(key); map.erase(iterator);
  • map[key]
  • map[key] = value;

unordered_map

unordered_map是一种无序关联容器，这些容器不是使用比较运算符来组织元素，而是使用一个哈希函数和关键字类型的==运算符。unordered_map底层的实现是哈希表

unordered_map的操作（增删查改）

  • unordered_map[key] = value; unordered_map.insert(pair<key,value>(key,value));
  • unordered_map.erase(key); unordered_map.erase(iterator);
  • map[key]
  • map[key] = value;

map与unordered_map之间的比较

  • map中元素有序，unordered_map中元素无序
  • map占用内存略低，unordered_map略高
  • 有顺序要求用map，对于查找问题一般用unordered_map
