1、显示日期与时间的命令：date

  • date：直接显示日期和时间
  • date +%Y/%m/%d：显示格式，年/月/日
  • date +%H:%M：显示格式，小时:分钟

2、显示日历的命令：cal

  • cal的基本语法：cal [[month] year]
  • cal：显示当前月份的月历
  • cal 2019：显示2019年的月历
  • cal 10 2019：显示2019年10月的日历

3、计算器：bc

4、数据同步写入磁盘：sync

5、关机：shutdown

  • shutdown -h now：立刻关机
  • shutdown -h 20:25：在20:25分关机
  • shutdown -r now：系统立刻重启
  • shutdown -r +10 “The system will reboot”：过10分钟系统重启，并发出告警信息

6、重启：reboot

7、切换执行等级：init

  • init 0：关机

8、改变文件属性与权限：

  • chgrp (-R) groupname dirname/filename：改变所属用户组
  • chown (-R) ownername:groupname dirname/filename
  • chmod (-R) xyz dirname/filename
    • r:4,w:2,x:1
  • chmod u/g/o/a +/-/= r/w/x

9、目录的相关操作

  • cd (绝对路径/相对路径)：切换目录
  • pwd (-P)：显示当前目录（加上参数表示非使用连接的路径）
  • mkdir (-mp) dirname：创建目录
    • m：配置文件夹的权限
    • p：直接将所需目录递归创建（创建多层目录）
  • rmdir (-p) dirname：删除目录
    • p：连通上层空目录一起删除
  • rm -r dirname：将目录连同目录下的所有东西都删除

10、查看文件与目录：ls

  • ls (-aAdfFhilnrRSt) dirname
  • ls (–color=(never,auto,always)) dirname
  • ls (–full-time) dirname
    • a：全部文件，包括隐藏文件
    • A：全部文件，包括隐藏文件，不包括（.和..两个目录）
    • d：仅列出目录本身，不列出目录内的文件数据
    • f：直接列出结果，不进行排序
    • F：根据文件、目录等信息给与附加数据结构
    • h：将文件容量以人类易读的方式列出来（human-readable）
    • i：列出inode号码
    • l：列出长数据串，包含文件的属性与权限
    • n：列出UID,GID，而非用户与用户组名称
    • r：将排序结构反向输出
    • R：连同子目录内容一起列出来
    • S：以文件容量大小排序
    • t：以时间排序
    • –color=never：不要依据文件特性显示颜色
    • –color=auto：默认设置
    • –color=always：显示颜色
    • –full-time：显示完整时间
    • –time=atime/ctime：输出访问时间或改变权限属性时间

11、复制命令：cp

  • cp (-adfilprsu) source_file destination_file
  • cp (options) source1 source2 source3… dirname
    • a：相当于-pdr的意思
    • d：若源文件为连接文件的属性，则复制连接文件属性而非文件本身
    • f：force，若已存在的目标文件无法打开，则删除后再尝试一次
    • i：若目标文件已存在，则覆盖前会确认操作
    • l：创建硬链接
    • p：连同文件属性一起复制
    • r：recursive，用于目录的复制行为
    • s：创建符号链接文件
    • u：当destination_file比source_file新的时候才覆盖更新
    • 如果source_file大于等于2，最后的目的文件必须是目录

12、移除文件或目录：rm

  • rm (-fir) file/directory
    • f：force，忽略不存在的文件，不会发出警告信息
    • i：interactive，询问是否进行删除
    • r：recursive，删除目录以及目录下的文件

13、移动文件或目录，重命名：mv

  • mv (-fiu) source destination
  • mv (options) source1 source2 … directory
    • f：force，若目标已存在，直接覆盖不询问
    • i：若目标存在，覆盖前进行询问
    • u：若目标文件存在，source较新时才进行覆盖

14、直接查看文件内容：cat，tac，nl

  • cat (-AbEnTv) filename
    • A：相当于-vET
    • b：列出行号，空白行不列
    • E：将结尾的断行字符$显示出来
    • n：打印出行号，空白号也列出来
    • T：将Tab键以^I显示出来
    • v：列出一些看不出来的特殊字符
  • tac：反向显示
  • nl （-bnw） filename
    • b：行号显示方式
    • n：列出行号的方式
    • w：行号字段占用的位数

15、可翻页查看：more，less

  • more filename
    • 空格键（space）：向下翻一页
    • Enter键：向下滚动一行
    • /字符串：向下查询字符串
    • :f：显示文件名以及目前显示的行数
    • q：退出
    • b：向回翻页
  • less filename
    • 空格键：向下翻一页
    • PageDown：向下翻一页
    • PageUp：向上翻一页
    • /字符串：向下查询字符串
    • ?字符串：向上查询字符串
    • n：重复前一个查询
    • N：反复重复前一个查询
    • q：退出

16、数据选取：head，tail

  • head (-n number) filename
    • number：从头部显示几行
  • tail (-n number) filename
    • number：从尾部开始显示几行

17、读取非文本文件：od

  • od (-t type) filename
    • a：使用默认字符输出
    • c：使用ASCII字符输出
    • d：使用十进制输出
    • f：使用浮点数输出
    • o：使用八进制输出
    • x：使用十六进制输出

18、修改文件时间或创建新文件：touch

  • touch (-acdmt) file
    • a：仅修改访问时间
    • c：仅修改文件的时间，若文件不存在则创建该文件
    • d：后接修改的日期
    • m：修改mtime
    • t：接修改后的时间

19、文件默认权限：umask

  • umask n：设置默认文件权限需要减掉的权限

20、修改、显示文件隐藏属性：chattr，lsattr

  • chattr (+-=)(ASacdistu) filename/dirname
    • i：使得文件不能被删除、改名，写入数据
  • lsattr (-adR) filename/dirname

21、查看文件类型：file

  • file filename

22、脚本文件名的查询：which

  • which (-a) command
    • a：将所有由PATH目录中找到的命令列出

23、文件名的查找：whereis，locate，find

  • whereis (-bmsu) file/dir
    • b：只查找二进制文件
    • m：只找在说明路径下的文件
    • s：只找source源文件
    • u：查找不同以上三个选项的其他文件
  • locate (-ir) keyword
    • i：忽略大小写的差异
    • r：后面可接正则表达式的显示方式
  • find (path) (option) (action)
